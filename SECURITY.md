# Security Policy

## Supported versions

Dunena is currently pre-1.0. Security fixes are applied to the latest release line.

## Reporting a vulnerability

Please report vulnerabilities privately.

1. Open a GitHub Security Advisory (preferred), or
2. Contact maintainers through private channel listed in repository settings.

Include:

- Affected version/commit
- Reproduction details
- Impact assessment
- Suggested mitigation (if known)

## Disclosure policy

- We will acknowledge reports within 72 hours.
- We aim to provide an initial triage within 7 days.
- Coordinated disclosure is expected. Please avoid public disclosure until a fix is available.
