## Quick Start

```bash
# Clone and start
git clone https://github.com/owenbellowen/dunena.git
cd dunena
bun run build:zig
bun run start

# Set a value
curl -X POST http://localhost:3000/cache/hello \
  -H "Content-Type: application/json" \
  -d '{"value": "world"}'

# Get it back
curl http://localhost:3000/cache/hello
```
